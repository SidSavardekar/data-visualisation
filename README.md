# data-visualisation
Data-visualisation Description
Rapid Response Visual: Infection Risk & Uncertainty Analysis
Project Overview

This Power BI dashboard provides a comprehensive analysis of infection risk levels and uncertainty estimates across different locations in Manchester. The objective is to help decision-makers identify high-risk areas, understand prediction uncertainty, and prioritize rapid response efforts.

Key Features
Risk categorization into High, Medium, and Low infection levels.
Geospatial visualization of infection hotspots across Manchester.
Infection risk distribution analysis using interactive charts.
Uncertainty assessment (Coefficient of Variation and Index of Dispersion).
Population and infection estimate summaries.
Interactive slicers for dynamic filtering and exploration.
Identification of priority zones requiring immediate intervention.
